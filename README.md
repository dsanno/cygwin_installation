cygwin_installation
===================

Scripts to install packages on cygwin.

# before using scripts

Install these packages:
tar, gnupg, gawk, wget

# scripts

* install_apt-cyg.sh  
Install apt-cyg
* install_ruby.sh  
Install ruby using rubyenv and ruby-build
* install_berkshelf.sh  
Install berkshelf
